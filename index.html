Text-Based Game:

This project is a text-based game made in Ruby that runs entirely in the console. It uses recursively called functions to simulate travelling between rooms of a house, trying to discover what has happened. Making this project was a fun way to experiment with Ruby and discover how we can turn methods and conditionals with user input into an interactive "game board".
